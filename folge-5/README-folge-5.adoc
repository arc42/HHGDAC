== Vereinfachung build.gradle
Wir haben für diese Folge wieder einen default-task für Gradle gesetzt -
das macht den build noch einfacher:

    cd folge-5
    ./gradlew

Falls Sie Gradle bereits installiert haben, geht's auch so:

    gradle
